Notable Projects

  • Arduino Racer Game | Github Link
    • Created a racing game designed for the Arduino which is displayed via a liquid crystal display. Implemented a multi-sync state machine design to tie together generation logic with physical button actuations.     • Can You Escape? is an immersive and challenging virtual reality escape room experience comprised of 1 intro room, 2 puzzle rooms, and one final room. I have designed and implemented all the puzzles (and mini puzzles) found within the game. Puzzles were designed to resemble the way a user may interact with an escape room in real life.
